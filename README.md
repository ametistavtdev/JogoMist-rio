# 🕵️ Mistério 50 Fases

Um jogo de investigação onde você resolve enigmas e descobre senhas para avançar!

## 🎮 Como Jogar

1. Leia o jornal com atenção
2. Analise as pistas encontradas
3. Digite a senha correta para desbloquear a próxima fase
4. Complete todas as 50 fases!

## 🔧 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📁 Estrutura

- `index.html` - Página principal
- `style.css` - Estilos visuais
- `script.js` - Lógica do jogo
- `fases.js` - Dados das 50 fases

## 🎯 Progresso

O progresso é salvo automaticamente no seu navegador!
